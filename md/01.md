## プロジェクトの作成
  1. プロジェクトの作成

  2. ターミナルでプロジェクトまで移動

  3. CocoaPodsの利用準備

      ```
      pod init
      ```

  4. テキストエディターでPodfileを編集
  
      `pod 'RealmSwift'`を追記

      ```
      # Pods for sampleRealmTodo
      pod 'RealmSwift'
      ```

  5. ライブラリをインストールする
  
      ```
      pod install
      ```

  6. プロジェクトを白いほうで開き直す
